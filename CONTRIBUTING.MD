# Contributing

We welcome contributions!

If you have any questions, [start a discussion](https://github.com/sustainableaviation/jetfuelburn/discussions) in this repository.

## Contributing to the Code

If you want to contribute to the package, please follow the [GitHub contribution workflow (fork, branch, PR)](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests).

### Code Style

Contributors must follow the [NumPy docstring style guide for documentation](https://numpydoc.readthedocs.io/en/latest/format.html) and [PEP 8](https://www.python.org/dev/peps/pep-0008/) for code style.
